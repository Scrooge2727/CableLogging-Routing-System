# Информационная система для решения задачи покрытия многоугольника

![Python](https://img.shields.io/badge/python-3.9+-blue.svg)

Данный репозиторий содержит программную реализацию интеллектуальной информационной системы, разработанной в рамках магистерской диссертации (ПетрГУ, ИМИТ, 2026 г.). 

Система предназначена для автоматизированного проектирования сетей канатных лесотранспортных установок на лесосеках со сложным рельефом и препятствиями.

## Основной функционал
- **Интерактивный графический интерфейс** для настройки параметров трассировки.
- **Модуль управления данными** с поддержкой импорта/экспорта топологии лесосеки в формате `.json`.
- **Три алгоритма маршрутизации**:
  1. *Базовый (веерный)* — детерминированный радиальный поиск.
  2. *Прямой генетический синтез* — стохастический поиск ломаных трасс.
  3. *Гибридный алгоритм* — комбинация пространственного генетического алгоритма и алгоритма Дейкстры на динамическом графе видимости (главное достижение работы, решающее проблему "мертвых зон").
- **Сбор статистики и экспорт**: автоматическая генерация отчетов, визуализация покрытия полигона и выгрузка координат опор для интеграции в ГИС/САПР.

Скачать скомпилированный `.exe` файл.

3. Запустите файл.
